1.What is creation phase and execution phase?

compiler runs the entire code for 2 times before actually executing the code and generates GEC(Global Execution context) inside this we have creation phase and execution phase.
Creation Phase:

  • In the first run it picks all the function declarations and stores them in memory with the reference.

  • In Second run,it picks all the variables and assign them as undefined.

    • Execution Phase:
  • Variables are asssigned with their values given the code.

  • Functions gets executed.

    3.What are function constructors?

    A constructor is a special function that creates and intializes an object instance of class.In JS constructor gets called when an object is created by using new keyword.we have multiple constructor Functions life arrow constructors for arrows, function constructors for functions and object constructors for objects etc.. when a constructor gets invoked the following sequence of operations takes place..

  • An empty object gets created
  • this keyword begins to refer to the new object and becomes to the current instance of the object.
  • New object is returned as the return value of the constructor.

    5.Explain prototypes

    In JS every object and function has a built in property called prototype.Prototype are the mechanism by which javascript objects inherit features from one another.The prototype itself an object and it will have its own prototype called prototype chain.The chain ends when we reach a prototype has null for its own prototype.In javascript Prototype property allows you to add new properties to object constructor. 

                    function Person(name,age){
                    this.name = name,
                    this.age = age
                }
                Person.prototype.nationality="Indian";
                const peson1 = new Person("yogendra", "23")
                console.log("name:"+ peson1.name,"age:"+peson1.age,peson1.nationality);

    6.What is prototype chain

    The Prototype object has prototype of its own and so on until it reaches to null as its prototype.So,linking this with another is called Prototype Chain.Null has no prototype and it acts as final link in this prototype chain.
